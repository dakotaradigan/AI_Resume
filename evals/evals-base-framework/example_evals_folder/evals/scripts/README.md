# Scripts

Code-based eval scripts and automation live here.

## Contents
- Code-based evals for objective/rule-based failure modes
- Runner scripts that execute eval suites
- Helper utilities for data loading and result formatting

## Naming
`eval_{criterion}.py` — e.g. `eval_schema_valid.py`, `eval_tool_calls.py`
`run_all.py` — Main runner that executes the full eval suite
